# VibeOut Audio Files

Place your audio files here in .ogg format:

## Required Files:
- `Drums1.ogg` - Calm background drums
- `Drums2.ogg` - Energetic beats for high vibe
- `Bass1.ogg` - Light bass line
- `Bass2.ogg` - Heavy bass for maximum vibe
- `Chords1.ogg` - Ambient chords
- `Chords2.ogg` - Rich harmonies
- `Melody1.ogg` - Lead melody for max vibe
- `Melody2.ogg` - Additional melody layer

## Audio Requirements:
- Format: OGG Vorbis (for best compatibility)
- Sample Rate: 44.1kHz
- Channels: Stereo
- Loop length: 8-16 bars for seamless looping
- Volume levels should be balanced for layering

## VibeOut Mode Behavior:
1. **MainMenu**: Drums1 + Bass1 (low volume)
2. **VibeLevel1**: Drums1 + Bass1 + Chords1
3. **VibeLevel2**: Drums2 + Bass1 + Chords1 (BEATS KICK IN!)
4. **VibeLevel3**: Drums2 + Bass2 + Chords2 + Melody1 (MAXIMUM VIBE!)

The VibeAudioManager will automatically layer these tracks based on search activity and good seed discovery!
